2. 곱셈 
   - $(a \times b)\mod c = (a \mod c \times b \mod c)\mod c$

3. 뺄셈(음수가 될 수 있기 때문)
    - $(a - b) \mod c = (a \mod c - b \mod c + c) \mod c$
